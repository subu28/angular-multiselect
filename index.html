Usage

<multiselect2 optionlist='optionlist' label='name' selectedlist='selectedlist'></ multiselect2>

optionlist : Array of objects to be passed as options.

label : The key to be used as label

selectedlist : Array of selected objects

Make sure that you use relevant CSS to customize the multiselect for your need. CSS on this page should prove helpfull.
